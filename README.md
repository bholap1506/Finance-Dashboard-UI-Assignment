# 💰 Finance Dashboard UI (FinDash)

A clean and interactive **Finance Dashboard UI** built using **HTML, CSS, and Vanilla JavaScript**.
This project demonstrates frontend development skills including UI design, state management, data handling, and user interaction.

---

## 🚀 Live Preview

>  https://bholap1506.github.io/Finance-Dashboard-UI-Assignment/


---

## 📌 Objective

The goal of this project is to build a simple dashboard interface that allows users to:

* View financial summaries
* Explore transactions
* Understand spending patterns
* Interact with UI based on user roles

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Custom styling + responsive design)
* JavaScript (Vanilla JS)
* Chart.js (for data visualization)

---

## ✨ Features

### 📊 Dashboard Overview

* Total Balance, Income, Expenses cards
* Transaction count
* Balance trend (Line Chart)
* Spending breakdown (Doughnut Chart)

---

### 📋 Transactions Section

* View all transactions with:

  * Date
  * Category
  * Type (Income / Expense)
  * Amount
  * Notes
* Features:

  * 🔍 Search
  * 🎯 Filter (Type & Category)
  * ↕️ Sorting (Date & Amount)

---

### 👤 Role-Based UI (Frontend Simulation)

* **Viewer**

  * Can only view data
* **Admin**

  * Can add and edit transactions

Role can be switched using a dropdown.

---

### 📈 Insights Section

* Highest spending category
* Monthly expense comparison
* Smart financial observation

---

### 🌙 Extra Features

* Dark Mode toggle
* Responsive design (Mobile + Desktop)
* Modal for adding/editing transactions
* Empty state handling
* Clean UI/UX

---

## 🧠 State Management

All application data is handled using a centralized `state` object:

* Transactions data
* Filters (search, category, type, sort)
* User role
* Theme (light/dark)
